# comfyui-ltx

## Virtual environment (Windows / PowerShell)

From this folder, activate the venv:

```powershell
.\venv\Scripts\Activate.ps1
```

If PowerShell blocks scripts, allow them for your user (one-time):

```powershell
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

## Run ComfyUI

```powershell
cd ComfyUI
# py main.py
py main.py
```
